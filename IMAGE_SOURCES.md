# Free High-Quality Home Health Care Images

## Best Free Stock Photo Sites

### 1. **Unsplash** (Recommended - Highest Quality)
- **URL**: https://unsplash.com
- **License**: Free for commercial use, no attribution required
- **Quality**: Professional, high-resolution photos
- **Search Terms**:
  - "home health care"
  - "nurse visit home"
  - "elderly care"
  - "healthcare professional"
  - "medical care home"
  - "nurse patient"
  - "home health aide"
  - "caregiver"
  - "healthcare worker"

### 2. **Pexels**
- **URL**: https://www.pexels.com
- **License**: Free for commercial use
- **Quality**: High-resolution, diverse collection
- **Search Terms**: Same as above

### 3. **Pixabay**
- **URL**: https://pixabay.com
- **License**: Free for commercial use
- **Quality**: Over 2.6 million images
- **Search Terms**: Same as above

## Recommended Image Placements

### Hero Section
- **Image Type**: Nurse or healthcare professional in a home setting
- **Style**: Warm, professional, compassionate

- **Style**: Diverse, professional, approachable

## Image Requirements

- **Resolution**: Minimum 1920x1080px (Full HD)
- **Format**: JPG or WebP
- **Aspect Ratio**: 
  - Hero: 16:9 or 21:9 (wide)
  - Cards: 4:3 or 1:1 (square)
  - Team: 3:4 or 1:1 (portrait/square)

## Download Instructions

1. Visit Unsplash.com (recommended for highest quality)
2. Search using the terms above
3. Click on an image you like
4. Click "Download free" button
5. Save to `/public/images/` folder in your project

## Image Optimization

After downloading, optimize images:
- Use tools like TinyPNG or Squoosh
- Convert to WebP format for better performance
- Maintain quality while reducing file size

## Example Image URLs (Direct Links)

You can also use Unsplash's API for direct image URLs:
- Format: `https://images.unsplash.com/photo-[ID]?w=1920&q=80`
- Or use their Source API for curated collections

## Recommended Collections

Search for these photographers on Unsplash who specialize in healthcare:
- Search: "healthcare" and filter by "Collections"
- Look for: "Medical", "Healthcare", "Nursing" collections
