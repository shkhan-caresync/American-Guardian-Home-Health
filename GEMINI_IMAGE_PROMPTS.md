# Gemini AI Image Generation Prompts

Copy and paste these prompts into Google Gemini to generate high-quality images for your website.

## Image Specifications
- **Aspect Ratio**: 16:9 for hero/wide images, 4:3 for cards, 1:1 for team photos
- **Style**: Professional, modern, bright, clean, healthcare-focused
- **Color Palette**: Bright whites, soft cyan/blue tones, warm lighting
- **Quality**: High resolution, photorealistic, professional photography style

## Usage Instructions

1. **Open Google Gemini**: Go to https://gemini.google.com
2. **Copy each prompt** above
3. **Paste into Gemini** and generate the image
4. **Download** the generated image
5. **Rename** the file to match the suggested file name
6. **Save** to `/public/images/` folder
7. **Optimize** images using TinyPNG or Squoosh (keep under 500KB)

## Tips for Best Results

- Generate each image multiple times and choose the best one
- If an image doesn't match the style, add "brighter" or "more modern" to the prompt
- Ensure diversity in representation across all images
- Make sure all images have the bright, clean, modern aesthetic
- Check that lighting is warm and natural

## Alternative: Batch Generation

You can also ask Gemini:
```
Generate 10 professional healthcare images for a modern home health care website. Style: bright, clean, modern, professional photography. Include: nurses, therapists, patients, home settings, diverse representation, warm lighting, cyan/blue color accents.
```

Then select the best ones for each section.
